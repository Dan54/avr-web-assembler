Windows computers need to have a driver set up to interface with the AVR microcontroller. You can set this up using the instructions here.

	sbi DDRD,4
loop:
    sbis PIND,7
    rjmp button_down
    cbi PORTD,4
    rjmp loop
button_down:
    sbi PORTD,4
    rjmp loop
